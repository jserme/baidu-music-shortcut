#百度音乐Chrome快捷键支持

##Why
对于一个习惯了`百度音乐`同时习惯了各种`快捷键`的人来说，用鼠标点来点去，实在是。。。很难过

##How
* 在页面插入两个脚本，调用百度音乐的一些变量来实现，恩，这个导致了我看了部分百度音乐的代码，还好它只是压缩了空格，没有压缩变量名
* 图标是用百度图片搜到的两个图片合成的
